Generar archivos de configuración

Recomendación, si se desea ejecutar Wellness Tool en una tablet o celular, primero generar esta configuración en un ordenador y luego copiar o enviar Wellness Tool a la tablet o celular.

Paso 1 - Descargar el archivo modelo para cargar la configuración

Descargar modelo

Descargar último import

Paso 2 - Editar archivo de Configuración

* Ajusta el archivo modelo a su gusto solapa a solapa.
* No modifique los nombres de las solapas.
* La columna edad en jugadores tiene una formula calculada por si agrega un nuevo jugador podria arrastrarla.

Paso 3 - Importar archivo de configuración

* Los archivos se generaran y debe guardarse en el directorio de "Descargas".
* Para no entrar en cambiar configuraciones según navegadores y sistemas operativos, debemos tomar esos archivos y guardarlos en el directorio "/data/json/" donde se encuentra el proyecto.

Paso 4 - Imagenes de jugadores

Si se desea cargar las fotos de los jugadores se deberían guardar en la carpeta "/img/players/" donde los nombres deberían ser el número de id con extencion "png", por ejemplo "1.png". Si tiene las fotos y quisiera quitarle el fondo y dejar solo la cara podria recurrir a remove.bg.
Sino puede importar una por una desde aquí que se guardaran en la carpeta "/img/players/".
